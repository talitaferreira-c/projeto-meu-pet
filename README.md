# projeto-meu-pet

Este é um projeto simples desenvolvido com HTML, CSS e JavaScript, com o objetivo de praticar conceitos básicos de desenvolvimento web.

## Sobre o projeto

O site apresenta informações sobre minha pet Nina, uma Spitz Alemão, incluindo:

- Descrição
- Rotina
- Imagem
- Interação com botão usando JavaScript

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Aprendizados

Neste projeto pratiquei:

- Estrutura HTML
- Estilização com CSS
- Manipulação básica com JavaScript (DOM)
